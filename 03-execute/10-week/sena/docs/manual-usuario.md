# Manual De Usuario

## Dashboard

Muestra indicadores generales:

- Total de aprendices.
- Matriculas activas.
- Matriculas condicionadas.
- Matriculas canceladas.
- Ultimas matriculas registradas.

## Gestion De Aprendices

Permite:

- Registrar aprendiz.
- Editar datos.
- Eliminar aprendiz.
- Buscar por documento, nombre, email, telefono o programa.

## Gestion De Matriculas

Permite:

- Crear matricula seleccionando un aprendiz existente.
- Editar ficha, jornada, estado y fechas.
- Eliminar matricula.
- Buscar por ficha, jornada, estado, documento o nombre del aprendiz.
