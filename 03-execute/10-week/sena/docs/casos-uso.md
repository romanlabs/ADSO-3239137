# Casos De Uso

## Registrar aprendiz

Actor: usuario administrativo.

Flujo:

1. Ingresa a Aprendices.
2. Completa documento, nombre, email, telefono y programa.
3. Guarda el formulario.
4. El sistema valida y registra el aprendiz.

## Registrar matricula

Actor: usuario administrativo.

Flujo:

1. Ingresa a Matriculas.
2. Selecciona un aprendiz.
3. Ingresa ficha, jornada, estado y fechas.
4. Guarda el formulario.
5. El sistema valida la relacion y registra la matricula.

## Consultar matriculas

Actor: usuario administrativo.

Flujo:

1. Ingresa a Matriculas.
2. Usa el buscador.
3. El sistema filtra resultados visibles.

## Actualizar estado

Actor: usuario administrativo.

Flujo:

1. Selecciona una matricula.
2. Presiona Editar.
3. Cambia el estado.
4. Guarda los cambios.
