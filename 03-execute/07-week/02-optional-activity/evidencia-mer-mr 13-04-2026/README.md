# Evidencia MER, MR y modelo físico en MySQL

## Contenido
- Diagrama MER
- Diagrama MR
- Script físico en MySQL

## Descripción
En esta evidencia se desarrolla el modelado de datos de un sistema dividido en tres dominios: ubicación, comercial y seguridad/acceso. Se construyó el Modelo Entidad Relación (MER), luego se transformó al Modelo Relacional (MR) y finalmente se elaboró el script físico en MySQL con sus llaves primarias, llaves foráneas y restricciones básicas.

## Estructura
- `/mer`: contiene el diagrama MER
- `/mr`: contiene el diagrama MR
- `/sql`: contiene el script SQL del modelo físico
